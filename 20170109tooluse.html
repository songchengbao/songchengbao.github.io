1、HBuilder

HBuilder总结


1、使用HBuilder新建项目

依次点击文件→新建→选择新建项目


2、使用HBuilder创建HTML页面

在项目资源管理器中选择刚才新建的项目,依次点击文件→新建→选择HTML文件


3、使用HBuilder边改边看试试查看编程效果

win系统按下Ctrl+P(MacOS为Command+P)进入边改边看模式,在此模式下,如果当前 打开的是HTML文件，每次保存均会自动刷新以显示当前页面效果


4、HBuilder代码块大量减少重复代码工作量

在打开的html页面中输入H,然后按下8,自动生成HTML的基本代码


2、WebStorm

WebStorm总结

欲先善其事，必先利其器

1、使用WebStorm特点总结

任何一个编辑器都需要保存(ctrl + s)，这是所有win平台上编辑类软件的特点， 但是webstorm编辑文件右上角是没有那个熟悉的 * 的。

好处：省去了ctrl + s之后，在结合Firefox的vim，基本不动鼠标就可以看到结果页面了。

坏处：没有以前的 * 标识，万一键盘误操作也会被立即存储。


2、使用WebStorm特点总结

任何一个编辑器只要文件关闭了就没有历史记录了，但是webstorm有。 vcs->Local History -> Show History

好处：只要webstorm不关闭，你的文件随时可以返回到之前的操作

坏处：webstorm关闭重启后这些历史记录就没有了；还有一个坏处就是由此带来的内存消耗也必然比较大。


3、使用WebStorm特点总结

任何一个编辑器，除了服务器svn之外，没有本地版本，但是webstorm提供一个本地文件修改历史记录

好处：相当于本地svn。

坏处：内存消耗也必然比较大。


4、使用WebStorm特点总结

集成了zencoding，HTML5，ftp，即时编辑（chrome）,自动完成,基于Mozilla的JavaScript调试器，JSLint、Less支持、 CoffeeScript支持、Node.js、单元测试、集成Git和svn版本控制等特性。


5、使用WebStorm特点总结

在编写CSS中，会智能的提示各种文件以及图片的路径，就不用再去确认这个文件是否存在了。


3、GitHub

GitHub 总结

1、GitHub 注册

要想使用 GitHub 第一步当然是注册 GitHub 账号：

1、首先打开https://github.com/pricing进行注册。

2、在打开的页面中点击绿色按钮「Sign up now」注册

3、在接下来的页面中创建用户名，填写 email 和设定密码， 点击「Create an account」按钮创建账户

2、GitHub 上创建一个新仓库

1、在任意的页面右上角点击+，然后点击新建仓库New repository.

2、为你的仓库创建一个简短便于记忆的名字。例如 “songchengbao.github.ol”。

3、为你的仓库添加一个描述（非必须的）。例如 “songchengbao Used to summarize”。

4、Public：公有仓库对于一个刚入门的新手来说是一个不错的选择。这些仓库在 GitHub 上对于每个人是可 见，你可以从协作型社区中受益。 （主要他不花钱 免费）

5、Private：私有仓库需要更多地步骤。它们只对于你来说是可用的，这个仓库的所有者属于你和你所指定要分 享的合作者。私有仓库仅仅对付费账户可用。（主要他花钱 不划算）

6、选择Initialize this repository with a README

7、点击Create repository

3、GitHub 提交你的第一个更改

1、在你仓库的文件列表，点击

2、在文件内容的上方，点击编辑按钮。

3、在需要更改的标签上，输入一些关于你自己的信息。

4、点击commit changes。

4、码云

码云 总结

1、码云 注册

要想使用 码云 第一步当然是注册 码云 账号：

1、首先是在码云平台注册，注册链接https://git.oschina.net。

2、依次填入各项，需注意的是：邮箱最好填写国内的邮箱，以免因为众所周知的原因无法 接收激活邮件，个性地址一经选定无法修改，请慎重填写.

3、然后，您的邮箱会收到一份激活邮件，请点击其中的链接激活账号，账号激活后， 流程就算完成了，接下来就是创建第一个项目了.

2、码云 创建项目

1、点击右上角的 "+" ，选择新建项目按钮，跳转到新建项目页面，依次填写各项信息然后点击创建就可以了， 这样我们一个项目就创建好了.

3、码云 本地与网络同步

1、在我的电脑里单击右键 检出 出现一个对话框，对话框第一栏需要填一个svn的地址（此地址去码云上找）

2、svn地址在头部标签质量分析下方 与绿色按钮Pull Request同行

3、单击svn 复制网址链接 再到电脑检出对话框粘贴 选择检出目录 确定即可！

4、如果你有要上传的文件 页面 可以打开检出目录 直接复制到里面 单击右键 选择TortoiseSVN按钮，会出现添加 添加成功后，再单击右键出现SVN提交 我们提交即可！

4、码云 找回本地文件

1、检出以后 我们如果误把东西删掉了 也可以找回 单击右键有更新两字 更新即可！

2、如果你把传上去的html页面删除了 同样可以更新回来！
