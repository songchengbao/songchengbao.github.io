JavasScript总结

一、JavasScript 数据类型

1、字符串型(String)

字符串是存储字符的变量,字符串可以是引号中的任意文本。您可以使用单引号或双引号.

2、数字(Number)

JavaScript 只有一种数字类型。数字可以带小数点，也可以不带；极大或极小的数字可以 通过科学（指数）计数法来书写。

3、数组(Array)

数组下标是基于零的，所以第一个项目是 [0]，第二个是 [1]，以此类推。

4、布尔(Boolean)

布尔只能有两个值：true 或 false，布尔常用在条件测试中！

5、对象(Object)

对象由花括号分隔,在括号内部对象的属性以名称和值对的形式来定义。属性由逗号分隔, 空格和换行无关紧要。声明可横跨多行！

6、空(null)

可以通过将变量的值设置为 null 来清空变量。


二、变量

1、常量声明

常量是公共的、静态的、不可改变的，必须初始值（一旦赋值，不可改变）

2、变量声明

变量是可变的，值不固定的，可分为成员变量、全局变量。


三、函数

1、函数语法

函数就是包裹在花括号中的代码块，前面使用了关键词 function 当调用该函数时，会执行函数 内的代码。

2、局部变量

在函数内部声明的变量（使用 var）是局部变量，所以只能在函数内部访问它。（该变量的作用域是局部的）。 你可以在不同的函数中使用名称相同的局部变量，因为只有声明过该变量的函数才能识别出该变量.

3、全局变量

在函数外声明的变量是全局变量，网页上的所有脚本和函数都能访问它。


四、日期

1、当前日期

使用 Date() 方法获得当日的日期。

2、定义日期

Date 对象用于处理日期和时间,可以通过 new 关键词来定义 Date 对象 Date 对象自动使用当前的日期和时间作为其初始值。


五、window

1、浏览器对象

浏览器对象模型 (BOM) 使 JavaScript 有能力与浏览器“对话”。

2、window对象

所有浏览器都支持 window 对象。所有 JavaScript 全局对象、函数以及变量 均 自动成为 window 对象的成员 全局变量是 window 对象的属性，全局函数是 window 对象的方法。甚至 HTML DOM的document也是 window对象的属性之一.
